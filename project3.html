Ice Cream App

Problem Space:

I had the desire to create an e-commerce mobile application similar to something like McDonald's, Taco Bell, etc. but in my own way.

Solution:

Decided on an ice cream ordering application that includes a shopping cart, accounts, item favoriting, and rewards systems.

My role:

  • UI & UX Designer
  • UX Researcher

Design Approach

I wanted to create an intuitive and simple application that lets users order items without feeling confused or overwhelmed.

Ideas for Features:

  • Home page with featured flavors
  • Search function with filters for item type (cone, bowl, etc.)
  • Account feature with ability to edit info
  • Add items to favorites
